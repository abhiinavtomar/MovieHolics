# MovieHolics
This is a movie searching platform with some special functionalities like following users, recommending them movies and also adding it to your watchlist.

## Getting Started

1.  Clone the repository on your system.
    Use __git clone https://github.com/abhiinavtomar/MovieHolics.git__ to clone repository.
2. Open the repository in favourite IDE and run npm install.
3. Run __node app.js__ to wait for the server to start , you will see __Server Started__ on your IDE terminal when the server gets started.
3. Open 'localhost:8080/' in your favourite browser and ENJOY.
